All about CSS

Introduction :

CSS stand for Cascading style sheet,it reffered to an CSS in a simple design language intended to simplify the process of making web page presentable.
CSS is a easy to learn and understand but it provide powerful control over the presentation of an HTML document.



Advantages :

  • CSS save time
  • Page load faster
  • Easy maintaince
  • Superior style to HTML
  • Multiple device compatability
  • Global web standards


Versions of css :

  1. CSS1 in December 1996
  2. CSS2 in May 1998
  3. CSS3 in June 1999


Maintainers of CSS :

CSS is created and mainted through a group of people within W3C called the CSS working group. The CSS working group crates a doucument called Specifications.When a specification has been discussed and officially realised by W3C members,it becomes a recommandations.



Type of CSS :

  1. 1) Embedded CSS
  2. 2) Inline CSS
  3. 3) External CSS
  4. 4) Imported CSS
